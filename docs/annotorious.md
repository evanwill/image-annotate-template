# Annotorious

This project uses Annotorious JavaScript library for adding annotations to images.
It uses the vanilla javascript approach described in the [Annotorious docs](https://annotorious.dev/guides/annotating-images/).

It generates annotation JSON files based on the [Annotorious Data Model](https://annotorious.dev/guides/data-model/), which follows the [W3C Web Annotation Data Model](https://www.w3.org/TR/annotation-model/).
